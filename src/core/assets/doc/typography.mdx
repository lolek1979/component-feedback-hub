## Usage

```jsx
// Font sizes
<h1 style={{ fontSize: 'var(--size-48)' }}>Heading 1</h1>
<p style={{ fontSize: 'var(--size-16)' }}>Body text</p>

// Font weights
<p style={{ fontWeight: 'var(--font-weight-700)' }}>Bold text</p>
<p style={{ fontWeight: 'var(--font-weight-400)' }}>Regular text</p>
```
