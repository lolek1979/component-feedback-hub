## Usage

```jsx
// In JSX with inline styles
<div style={{ margin: 'var(--spacing-s)' }}>
  <p style={{ padding: 'var(--spacing-xs)' }}>Content with spacing</p>
</div>

// In CSS
.container {
  margin-bottom: var(--spacing-m);
  padding: var(--spacing-s);
  gap: var(--spacing-xs);
}
```

## Guidelines

- Use spacing tokens instead of arbitrary pixel values
- For most UI elements, use smaller spacing values (5xs-xs) for internal spacing
- Use medium spacing values (s-xl) for layout and component spacing
- Reserve larger spacing values (2xl-3xl) for major section divisions
