# Icons
---

Icons from the **VZP NIS Assets** collection. <br/>
It is possible to change the color through the `className` property.

## Usage

```js
<IArrowDown />
<IArrowDown className="icon_blue" />
<IArrowDown className="icon_blue" style={{ width: 32, height: 32 }} />
```

## Examples

<div className="icon-gallery">
  <IconGallery>
    {Object.entries(Icons).map(([name, Icon]) => (
      <IconItem name={name} key={name}>
        <Icon />
      </IconItem>
    ))}
  </IconGallery>
</div>
