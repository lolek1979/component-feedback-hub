# CscHeaderButtons

The `CscHeaderButtons` component provides a set of header buttons for the CSC application. It includes a primary button that toggles a dropdown menu with options to create a new codebook or import an existing codebook. Additionally, there is a secondary button for settings. The component is designed to be flexible and responsive, making it suitable for various screen sizes and use cases.
