# CscFormModal

The `CscFormModal` component is a modal dialog used for creating or editing CSC (Codebook) entries. It provides a form with various input fields to capture the necessary information for a CSC entry. The form includes fields for the name, identifier, description, garant (responsible person), validity dates, and type of the CSC entry.

## Usage

The `CscFormModal` component can be used to create or edit CSC entries. It includes the following features:

- **Name**: A text input field for the name of the CSC entry.
- **Identifier**: A text input field for the identifier of the CSC entry.
- **Description**: A textarea for describing the purpose and usage of the CSC entry.
- **Garant**: A select input for choosing the responsible person (garant) from a list.
- **Validity Dates**: Date pickers for selecting the start and end dates of the CSC entry's validity.
- **Type**: Radio buttons for selecting the type of the CSC entry (e.g., internal, external).

The modal can be opened by clicking a button, and it includes a submit button to save the form data. The form data is validated before submission, and an alert is shown if required fields are missing.
