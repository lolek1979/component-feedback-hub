# CSCFilter

The CSCFilter component is a complex filter interface that combines various input elements for filtering data. It provides a flexible layout that adapts to different screen sizes.

## Features
- Responsive layout that adjusts to screen width
- Combines multiple filter inputs in a single row
- Additional actions row for supplementary controls
- Mobile-friendly design that stacks elements vertically below 900px

## Usage

The CSCFilter component is designed to be used in data-heavy interfaces where complex filtering is required. It maintains usability across different screen sizes by reorganizing its layout responsively.

### Desktop View
- Filter elements are arranged horizontally
- Actions row appears inline with main filters

### Mobile View (< 900px)
- Filter elements stack vertically
- Actions row moves below the main filters
- Full width utilization for better touch interaction
