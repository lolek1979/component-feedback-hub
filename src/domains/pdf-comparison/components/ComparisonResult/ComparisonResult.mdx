# ComparisonResult

The ComparisonResult component displays the result of a comparison operation with different visual states for success and error scenarios.

## Features
- Two variants: success and error
- Large icons for clear visual feedback
- Optional retry functionality for error states
- Responsive design
- Support for dark theme

## Usage

The ComparisonResult component is designed to provide clear feedback about the status of document comparison operations. It uses large, recognizable icons and clear messaging to communicate the result to users.

### Success State
- Green checkmark icon
- Positive messaging
- Light green background with green border

### Error State
- Red X icon
- Error messaging
- Light red background with red border
- Optional retry button
