# PdfComparisonHeader

The `PdfComparisonHeader` component displays the match percentage for PDF comparison results. It's designed to show users how similar two PDF documents are after comparison.

## Usage

```tsx
import { PdfComparisonHeader } from '@/domains/pdf-comparison/components';

function ComparisonResults() {
  return (
    <PdfComparisonHeader
      matchPercentage={85.5}
      id="comparison-header"
    />
  );
}
```
