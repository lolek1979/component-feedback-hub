# DocumentComparisonSection

The `DocumentComparisonSection` component displays a side-by-side comparison of document sections, perfect for comparing differences between two versions of a document.

## Features

- **Side-by-side layout**: Displays left and right document sections in a grid layout
- **Flexible content**: Supports titles, subtitles, and main content areas
- **Page information**: Optional page/section identifier
- **Responsive design**: Adapts to mobile screens by stacking sections vertically
- **Design system integration**: Uses standardized spacing, colors, and border radius tokens

## Design Tokens Used

- **Color**: `--table-primary` for background color
- **Gap**: `--spacing-xs` for consistent spacing between elements
- **Border Radius**: `--radius-s` for rounded corners
- **Padding**: `--spacing-m` for internal spacing

## Usage

```tsx
import { DocumentComparisonSection } from '@/design-system/molecules';

<DocumentComparisonSection
  id="comparison-1"
  leftTitle="Odeslaná"
  rightTitle="Podepsaná"
  leftSubtitle="MUDr. Petr Pouček, MBA, ředitel"
  rightSubtitle="MUDr. Jan Nezbeda, ředitel úseku"
  leftContent="Original document content..."
  rightContent="Modified document content..."
  pageInfo="Stránka 1"
/>
```

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `id` | `string` | ✅ | Unique identifier for the component |
| `leftTitle` | `string` | ✅ | Title for the left document section |
| `rightTitle` | `string` | ✅ | Title for the right document section |
| `leftContent` | `string` | ✅ | Content for the left document section |
| `rightContent` | `string` | ✅ | Content for the right document section |
| `leftSubtitle` | `string` | ❌ | Optional subtitle for the left section |
| `rightSubtitle` | `string` | ❌ | Optional subtitle for the right section |
| `pageInfo` | `string` | ❌ | Optional page or section information |
| `className` | `string` | ❌ | Additional CSS classes |

## Accessibility

- Uses semantic heading elements for titles
- Proper contrast ratios for text readability
- Responsive design for various screen sizes
