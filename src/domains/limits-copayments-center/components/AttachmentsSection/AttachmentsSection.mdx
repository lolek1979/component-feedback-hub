# AttachmentsSection

A comprehensive section component for file attachments that combines a header with title, badge, and description with the AttachmentsUpload component. The badge automatically displays the count of uploaded files.

## Features

- **Dynamic badge**: Badge automatically updates with the count of uploaded files
- **Header section**: Includes title, badge, optional text, and description
- **File upload**: Integrates AttachmentsUpload component for file management
- **Customizable**: Configurable file limits, size restrictions, and accepted file types
- **Flexible styling**: Badge color can be customized based on state

## Usage

```tsx
import { AttachmentsSection } from '@/domains/limits-copayments-center/components/AttachmentsSection';

function MyComponent() {
  const [files, setFiles] = useState<File[]>([]);

  const handleFilesChange = (selectedFiles: File[]) => {
    setFiles(selectedFiles);
  };

  return (
    <AttachmentsSection
      title="Přílohy"
      badgeColor="gray"
      optionalText="(nepovinné)"
      description="Nahrajte účtenky, faktury nebo plné moci."
      onFilesChange={handleFilesChange}
      maxFiles={5}
      maxFileSize={10}
    />
  );
}
```
