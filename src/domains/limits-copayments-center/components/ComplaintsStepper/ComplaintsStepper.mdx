# ComplaintsStepper

A horizontal stepper component for displaying progress through complaint creation steps.

## Features
- Horizontal step progression display
- Active/inactive step states
- Numbered step indicators
- Step separators
- Responsive typography
- Accessible design

## Props

| Prop | Type | Description |
|------|------|-------------|
| `steps` | `Step[]` | Array of step objects |

### Step Object

```typescript
interface Step {
  label: string;      // Step label text
  isActive: boolean;  // Active state flag
}
```

## Usage

```tsx
const steps = [
  { label: 'Complaint Type', isActive: true },
  { label: 'Subject', isActive: false },
  { label: 'Details', isActive: false },
  { label: 'Attachments', isActive: false },
  { label: 'Review', isActive: false },
];

<ComplaintsStepper steps={steps} />
```

## Visual Structure

- Each step displays a numbered circle and label
- Active steps are highlighted with brand color
- Inactive steps use secondary text color
- Steps are separated by arrow icons
- Numbers are displayed in circles (1, 2, 3, etc.)
