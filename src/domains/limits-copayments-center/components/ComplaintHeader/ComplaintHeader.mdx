# ComplaintHeader

A header component for displaying complaint information with breadcrumbs, status badges, and action buttons.

## Features
- Display complaint ID and insurance number
- Optional breadcrumbs navigation
- Status badges (registered)
- Action buttons for draft and registered complaints
- Opinion modal integration
- Responsive design
- Dark mode support

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `complaintId` | `string` | - | Complaint identifier |
| `firstName` | `string` | - | First name (optional) |
| `lastName` | `string` | - | Last name (optional) |
| `name` | `string` | - | Full name (optional) |
| `insuranceNum` | `string` | - | Insurance number |
| `showBreadcrumbs` | `boolean` | `false` | Show breadcrumbs navigation |
| `breadcrumbTitle` | `string` | `''` | Breadcrumb text |
| `onBreadcrumbClick` | `() => void` | - | Breadcrumb click handler |
| `complaintStatus` | `'registered'` | - | Complaint status |
| `isDraft` | `boolean` | `false` | Draft mode flag |
| `onApprove` | `() => void` | - | Approve action handler |
| `onRequestOpinion` | `() => void` | - | Request opinion handler |
| `onReject` | `() => void` | - | Reject action handler |
| `onSubmitForReview` | `() => void` | - | Submit for review handler |
| `onCancel` | `() => void` | - | Cancel action handler |
| `actionsDisabled` | `boolean` | `false` | Disable all actions |
| `currentStep` | `number` | `0` | Current step number |

## Usage

```tsx
<ComplaintHeader
  complaintId="C-12345"
  insuranceNum="1234567890"
  showBreadcrumbs
  breadcrumbTitle="Back to list"
  onBreadcrumbClick={() => router.back()}
  complaintStatus="registered"
  onApprove={handleApprove}
  onRequestOpinion={handleRequestOpinion}
  onReject={handleReject}
/>
```
