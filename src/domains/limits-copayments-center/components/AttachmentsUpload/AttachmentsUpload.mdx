# AttachmentsUpload

A specialized file upload component for attachments that allows users to upload multiple files with drag-and-drop support. The component validates file types, size limits, and displays a list of uploaded files with the ability to remove them.

## Features

- **Multiple file upload**: Support for up to 5 files by default
- **Drag and drop**: Files can be dragged and dropped into the upload area
- **File validation**: Validates file types (PDF, JPG, PNG, GIF) and size limits (10MB default)
- **File list**: Displays uploaded files with name and size
- **Remove files**: Individual files can be removed from the list
- **Localized**: All texts are localized using next-intl
- **Error handling**: Shows validation errors for unsupported formats or size limits

## Usage

```tsx
import { AttachmentsUpload } from '@/domains/limits-copayments-center/components/AttachmentsUpload';

function MyComponent() {
  const handleFilesChange = (files: File[]) => {
    console.log('Selected files:', files);
  };

  return (
    <AttachmentsUpload
      onFilesChange={handleFilesChange}
      maxFiles={5}
      maxFileSize={10}
      acceptedFileTypes={['.pdf', '.jpg', '.jpeg', '.png', '.gif']}
    />
  );
}
```
