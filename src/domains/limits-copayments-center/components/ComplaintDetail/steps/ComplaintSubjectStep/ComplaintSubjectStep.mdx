# ComplaintSubjectStep

A step component in the complaint creation flow for selecting complaint subject.

## Features
- Step title display
- Navigation buttons (back and continue)
- Integration with complaint store
- Responsive design

## Usage

```tsx
<ComplaintSubjectStep />
```

## Integration

This component uses the `useComplaintStore` hook to manage step navigation:
- `nextStep()` - Navigate to the next step
- `previousStep()` - Navigate to the previous step
