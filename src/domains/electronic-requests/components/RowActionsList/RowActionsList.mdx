# RowActionsList

The `RowActionsList` component displays a list of actions that can be performed on a row in a table. The actions available depend on the status of the row, which can be 'Awaiting', 'Draft', 'Active', or 'Inactive'. Each action is represented by an icon and a label, and clicking on an action triggers a callback function with the action's label as an argument.
