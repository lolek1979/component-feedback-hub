# InlineMessage

The `InlineMessage` component is used to display brief, inline messages to the user. These messages can convey different types of information, such as success, warning, error, or informational messages.

<Canvas of={Default} />

## Usage

The `InlineMessage` component can be used to provide feedback to the user in various scenarios. It supports different message types, which can be customized using the `type` prop.
