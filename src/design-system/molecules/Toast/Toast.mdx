# Toast

The Toast component is used to display short notifications to the user. It supports several variants (success, error, info, warning) and can be triggered using the `toast` hook.

It is based on the [Sonner](https://www.npmjs.com/package/sonner) package, which provides a simple and customizable toast notification system.

## Examples

### Default

<Canvas of={Default} />

You can trigger a toast using the `toast.success`, `toast.error`, `toast.info`, or `toast.warning` functions.

```tsx
import { toast } from './Toast';

toast.success('Saved!');
toast.error('An error occurred!');
toast.info('Information.');
toast.warning('Warning!');
```

### Custom Duration

<Canvas of={WithCustomDuration} />

```tsx
import { 
  
} from './Toast';

toast.info('Eight seconds duration toast', { duration: 8000 });
```

### Infinity Duration

<Canvas of={WithInfinityDuration} />

```tsx
import { toast } from './Toast';

toast.info('Infinity duration toast', { duration: Infinity });
```

### With action

<Canvas of={WithAction} />

```tsx
import { toast } from './Toast';

toast.error('error toast', {
  action: {
    label: 'Retry',
    onClick: () => alert('retry clicked'),
  },
});
```
