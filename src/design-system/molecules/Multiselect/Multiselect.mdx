# Multiselect

---

Multiselect is a wrapper around the Select component that allows users to select multiple items from a collection. It includes search functionality, smart text display with truncation, and action buttons.

## Features

- **Search functionality**: Built-in search to filter options
- **Smart text display**: Shows selected items with intelligent truncation and count badge
- **Action buttons**: "Select all" and "Clear filter" buttons for quick selection management
- **Accessibility**: Full keyboard navigation and screen reader support
- **Validation**: Helper text and error state support
