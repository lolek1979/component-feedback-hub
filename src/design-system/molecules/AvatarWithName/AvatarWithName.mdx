# UserAvatarWithName

 UserAvatarWithName component displays a user's avatar along with their name, organization, and email.
 or just the name if displayNameOnly param is true.
 If the input string does not match the expected format e.g. "First name Surname (Organization), email", 
 it returns a single dash '-'.
