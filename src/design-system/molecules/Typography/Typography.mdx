## Usage Guidelines

- **Font Family**: Always uses Inter font family
- **Color**: Always inherits color from parent - never hardcoded
- **Semantic HTML**: Use appropriate HTML elements (h1-h6 for headings, p for paragraphs, etc.)
- **Link Variants**: Use link variants for interactive text elements
- **Consistency**: Stick to defined variants to maintain design consistency

## Design Tokens

The component uses CSS design tokens for:
- Font sizes: `--size-*` variables
- Font weights: `--font-weight-*` variables
- Consistent spacing and line heights
- Text decoration properties for links

## Accessibility

- Uses semantic HTML elements when specified via the `component` prop
- Inherits color for proper contrast compliance
- Supports all standard HTML attributes for accessibility (aria-*, role, etc.)
- Link variants include proper underline styling for accessibility
