# BugReportModal

The BugReportModal component displays a modal dialog that allows users to report bugs or issues they encounter in the application.

## Usage

The BugReportModal is controlled by the `bugReport` URL query parameter. When this parameter is set to `true`, the modal will be visible, allowing users to submit bug reports.

```jsx
import BugReportModal from '@/components/organisms/BugReportModal';

const YourComponent = () => {
  return (
    <>
      {/* Other components */}
      <BugReportModal />
    </>
  );
};
```
