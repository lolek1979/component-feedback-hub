# ScreenshotSnackbar

The ScreenshotSnackbar component displays a control bar that appears when a user initiates a screenshot capture process for bug reporting.

## Usage

The ScreenshotSnackbar is controlled by the `captureScreenshot` URL query parameter. When this parameter is set to `true`,
the snackbar will be visible, prompting users to either proceed with taking a screenshot or cancel the operation.

```jsx
import ScreenshotSnackbar from '@/components/organisms/ScreenshotSnackbar';

const YourComponent = () => {
  return (
    <>
      {/* Other components */}
      <ScreenshotSnackbar />
    </>
  );
};

```
