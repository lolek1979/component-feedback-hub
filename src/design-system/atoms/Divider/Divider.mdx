# Divider

The `Divider` component is used to visually separate content in a layout. It can be oriented horizontally or vertically and can have different variants such as "primary", "subtle", "dotted", or "prominent". The component is highly customizable with additional class names and styles.

.divider {
  background-color: var(--border-primary);

    height: var(--units-1);

    width: var(--units-1);

    background-color: var(--border-primary);
  }

  &.dividerSubtle {
    background-color: var(--border-subtle);
  }

  &.dividerProminent {
    background-color: var(--border-prominent);
  }

  &.dividerDotted {
    background-color: transparent;

    &.dividerHorizontal {
      height: 0;
      display: block;
      border-top-width: var(--border-s);
      border-top-style: solid;
      border-image: repeating-linear-gradient(
          to right,
          var(--border-subtle) 0,
          var(--border-subtle) var(--units-2),
          transparent var(--units-2),
          transparent var(--units-4)
        )
        1;
    }

    &.dividerVertical {
      width: 0;
      display: block;
      border-left-width: var(--border-s);
      border-left-style: solid;
      border-image: repeating-linear-gradient(
          to bottom,
          var(--border-subtle) 0,
          var(--border-subtle) var(--units-2),
          transparent var(--units-2),
          transparent var(--units-4)
        )
        1;
    }
