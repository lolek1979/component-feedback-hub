# DarkmodeToggle

The Dark Mode Toggle component lets users switch themes by applying a `.dark` class to the `<body>`, enabling dark theme styling across components by updating local CSS variables.

### Default

## Usage Example
To make a component responsive to dark mode, define CSS variables in the component's CSS Module:

```css
/* YourComponent.module.css */
.wrapper {
  --wrapper-background-color: var(--color-white);
  background-color: var(--wrapper-background-color);
}

:global(.dark) .wrapper {
  --wrapper-background-color: var(--color-black-950);
}
```

When the .dark class is added to `<body>`, the background color changes to `--color-black-950`. 
The component uses `background-color: var(--wrapper-background-color);` to update automatically based on the theme toggle.
